## 🔗 交互流程

### 主流程: 录入错题
```
首页 → 点击"快速录入" → 拍照/上传
  → OCR识别预览 → 确认内容
  → 选择科目年级 → 填写错误答案
  → 点击"AI分析" → 查看分析结果
  → 保存到错题本
```

### 复习流程
```
首页 → 点击"错题本" → 筛选今日复习
  → 选择错题 → 遮挡答案重做
  → 提交答案 → 查看正误
  → 标记掌握程度
```
