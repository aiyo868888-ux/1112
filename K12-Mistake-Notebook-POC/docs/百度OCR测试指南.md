## 🧪 测试步骤

### 1. 打开前端应用

在浏览器中访问: **http://localhost:5173**

### 2. 进入录入页面

点击 **"录入错题"** 按钮

### 3. 上传测试图片

**推荐测试图片类型**:
- ✅ 试卷题目（印刷体）
- ✅ 教辅资料（印刷体）
- ✅ 作业答案（印刷体）
- ⚠️ 手写内容（识别率会降低）

**图片要求**:
- 格式: JPG, PNG
- 大小: < 10MB
- 清晰度: 越清晰越好
- 建议: 分辨率 > 300dpi

### 4. 执行OCR识别

点击 **"🔄 重新识别"** 按钮

**预期结果**:
- 等待1-3秒
- OCR文本框中会显示识别出的文字内容
- **这次应该是真实的识别结果，不再是模拟数据**

### 5. 查看识别结果

**识别成功示例**:
```
解方程 x² + 2x + 1 = 0

已知一元二次方程：x² + 2x + 1 = 0
求：方程的解

解：
x² + 2x + 1 = 0
(x + 1)² = 0
x + 1 = 0
x = -1
```

### 6. 测试AI分析（可选）

点击 **"🤖 AI分析"** 按钮查看智能分析结果

## ⚠️ 常见问题

### Q1: 识别结果不准确怎么办？

**A**:
1. 确保图片清晰（避免模糊、反光）
2. 确保文字水平（避免倾斜）
3. 使用高分辨率图片
4. 如果是手写体，识别率会降低是正常的

### Q2: 提示"每日流量超限"怎么办？

**A**:
- 说明免费额度已用完
- 可以购买次数包或按量付费
- 或者等待下月1日重置

### Q3: 识别速度慢？

**A**:
- 正常速度: 1-3秒
- 首次调用可能较慢（需要获取token）
- 后续调用会快很多

### Q4: 如何查看已经用了多少次？

**A**:
- 登录百度智能云控制台
- 访问OCR服务概览页
- 查看"调用量统计"
