## 🔬 测试场景

### 场景1: 数学错题录入
**目标**: 录入一道初二数学错题

**步骤**:
1. [ ] 打开App，记录启动时间
2. [ ] 点击"录入错题"按钮
3. [ ] 拍照/上传错题图片
4. [ ] 等待OCR识别，记录识别准确率
5. [ ] 编辑识别结果
6. [ ] 添加错因标签
7. [ ] 保存到错题本

**评分指标**:
- 操作步骤数: ___
- 完成时间: ___ 秒
- OCR准确率: ___ %
- UX满意度: ___/5

### 场景2: 查看错题分析
**目标**: 查看AI错因诊断

**步骤**:
1. [ ] 进入错题详情页
2. [ ] 查看错因分析
3. [ ] 查看同类题推荐
4. [ ] 尝试添加笔记

**评分指标**:
- 错因分析深度: ___/5 (1=简单标签, 5=详细诊断)
- 同类题匹配度: ___/5
- 笔记功能易用性: ___/5

### 场景3: 复习提醒
**目标**: 使用复习功能

**步骤**:
1. [ ] 查看今日复习任务
2. [ ] 完成1道错题重做
3. [ ] 查看复习提醒设置

**评分指标**:
- 提醒机制有效性: ___/5
- 复习流程流畅度: ___/5

## 📹 录制工具设置

### iOS
```
设置 → 控制中心 → 添加"屏幕录制"
开始录制: 控制中心 → 录制按钮
```

### Android
```
下载: SCR Screen Recorder (Google Play)
设置: 分辨率1080p + 帧率30fps
```
