## 🛠️ 技术栈

### 后端
- **OCR**: PaddleOCR (文字) + LaTeX-OCR (公式)
- **AI**: Qwen2.5-7B (Ollama本地)
- **API**: FastAPI
- **向量**: ChromaDB

### 前端
- **框架**: React 18
- **构建**: Vite
- **样式**: TailwindCSS
- **HTTP**: Axios

### 基础设施
- **语言**: Python 3.10+, Node.js 18+
- **版本管理**: Git
- **AI运行时**: Ollama
