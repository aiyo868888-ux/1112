## 🎯 现在的行动 (今晚/明天)

### A. 启动技术POC (15分钟)

```bash
# 1. 进入项目目录
cd K12-Mistake-Notebook-POC

# 2. 运行启动脚本
# Windows:
quickstart.bat

# Mac/Linux:
chmod +x quickstart.sh
./quickstart.sh
```

**安装Ollama** (如果未安装):
1. 访问 https://ollama.ai
2. 下载并安装
3. 运行: `ollama run qwen2.5:7b`

**启动3个服务** (新开3个终端窗口):
```bash
# 终端1: OCR服务
python ocr_service/app.py

# 终端2: AI服务
python ai_service/app.py

# 终端3: 前端
cd frontend && npm run dev
```

**访问**: http://localhost:5173
