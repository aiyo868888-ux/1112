# 📱 K12错题本 Android应用

## 🎯 项目概述

基于Web版K12错题本成功经验，开发Android原生应用，实现拍照→选题→保存→组题的完整流程。

---

## 📱 核心功能

### 1. 📷 拍照录入
- 使用CameraX实现相机功能
- 自动对焦和优化
- 支持前后摄像头
- 图片保存到本地缓存

### 2. 📐 AI选题
- **自动检测**: AI自动识别题目边界
- **手动调整**: 支持手动添加/删除/调整框选
- **多题支持**: 一次识别多个题目
- **智能提示**: 检测进度和结果提示

### 3. ✅ 保存错题
- OCR文字识别（调用后端API）
- 填写错误答案和正确答案
- 选择科目和年级
- AI智能分析（三维分析）

### 4. 📚 组题管理
- 按科目分类
- 按日期排序
- 搜索和筛选
- 复习状态标记
- 统计信息展示

---

## 🏗️ 技术架构

### 技术栈
- **语言**: Kotlin
- **UI框架**: Jetpack Compose
- **架构**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **相机**: CameraX
- **数据库**: Room
- **网络**: Retrofit + OkHttp

### 最低要求
- **Android版本**: API 24 (Android 7.0)
- **目标SDK**: API 34 (Android 14)
- **内存**: 2GB+
- **存储**: 100MB

## 🔄 完整流程

```
打开应用 → 主页 → 拍照录入 → 📷拍照
→ AI检测题目 → 调整框选 → ✅保存
```
